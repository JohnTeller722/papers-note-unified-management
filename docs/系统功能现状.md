# 系统功能现状（截至 2026-04-15）

> 本文用于快速说明：当前已经实现什么、部分实现到什么程度、哪些能力仍未实现。

## 1. 已实现功能

### 1.1 模板体系（粗读 / 精读）

- 已支持粗读模板与精读模板分离管理，并可创建多套模板。
- 已支持精读模板基于粗读模板继承字段（精读在粗读基础上扩展）。
- 已支持模板新增、重命名、删除（软删除 `is_active=0`）。
- 相关接口：
  - `GET /api/templates`
  - `POST /api/templates`
  - `PUT /api/templates/:id`
  - `DELETE /api/templates/:id`

### 1.2 文献录入与编辑

- 录入页支持：
  - 论文基础字段（题目/年份/会议/作者/链接/MD路径）
  - 模板动态字段填写
  - 粗读/精读阶段切换
- 保存后已实现：
  - 弹窗提示“保存成功”
  - 自动清空表单
- 记录中心支持编辑已存记录并回填到录入页。
- 相关接口：
  - `POST /api/papers`
  - `PUT /api/papers/:id`
  - `GET /api/papers/:id`

### 1.3 粗读升级精读

- 已支持对同一条记录执行 `rough -> deep` 升级。
- 升级行为记录到 `paper_transitions`。
- 相关接口：`POST /api/papers/:id/upgrade`

### 1.4 PDF 本地解析（元数据自动填充）

- 录入页支持拖拽/选择 PDF。
- 后端基于 `pdf-parse` 在本地提取：题目、作者、年份、DOI（尽力提取）。
- 前端会自动回填到表单（若对应输入为空）。
- 相关接口：`POST /api/pdf/parse`

### 1.5 记录中心（检索、筛选、批量操作）

- 支持关键词筛选 + 阶段筛选 + 优先级/类别/关系筛选。
- 支持全选/清空选择。
- 支持单条操作：编辑、升级精读、打开MD、删除。
- 相关接口：
  - `GET /api/papers`
  - `DELETE /api/papers/:id`

### 1.6 下拉选项管理（可配置枚举）

- 已支持对三类枚举组维护：
  - 相关工作类别 `related_category`
  - 精读优先级 `reading_priority`
  - 和我的关系 `relation_type`
- 支持新增、编辑、启停用、删除。
- 相关接口：
  - `GET /api/enums`
  - `POST /api/enums/:groupKey/options`
  - `PUT /api/enums/:groupKey/options/:optionId`
  - `DELETE /api/enums/:groupKey/options/:optionId`

### 1.7 导出（Excel）

- 支持导出所选记录。
- 支持按当前筛选结果导出。
- 导出格式 `.xlsx`，并生成下载链接。
- 相关接口：`POST /api/export`

### 1.8 MD 笔记联动（创建 + 打开）

- 已支持“立即创建MD”：按标题/年份自动创建本地 `.md` 文件并回填路径。
- 保存文献时支持“自动创建MD”（若未手填路径）。
- 已支持“打开MD”：
  - 路径为空时提示错误
  - 文件不存在时自动创建再打开
  - 仅允许 `.md/.markdown`
- 相关接口：
  - `POST /api/papers/create-md`
  - `POST /api/papers/open-md`
  - `GET/PUT /api/settings`（配置 notesDir）

### 1.9 首页仪表盘（独立首页）

- 已新增独立首页（Home），并作为默认启动页。
- 已实现三大区块：
  - 最近阅读（前 5 条，支持编辑/跳转记录中心）
  - 精读候选（粗读 + A/B 优先级，支持编辑/升级精读）
  - 导出快捷入口（导出最近阅读 / 导出精读候选 / 导出当前筛选结果）
- 已新增“论文数量仪表盘”可视化（速度盘风格）：
  - 显示总论文数
  - 显示粗读/精读数量
  - 显示候选率
  - 指针按数量占比动态映射

### 1.10 UI 与主题

- 顶部栏已改为侧边栏导航（首页 / 录入页 / 记录中心 / 设置）。
- 已支持日夜主题切换，并使用 `localStorage` 记忆。
- 夜间模式下下拉框与选项面板已适配深色背景与文字。

## 3. 未实现功能（相对需求方案）

### 3.1 模板字段高级能力

- 未实现字段级排序拖拽、字段分组、字段开关版本管理。
- 当前模板管理以基础字段新增/改名/删除为主。

### 3.2 草稿恢复与异常恢复

- 尚未实现录入草稿自动持久化与崩溃恢复。

### 3.3 多用户/同步/协作

- 当前为本地单机单用户。
- 未实现账号体系、权限控制、多设备同步。

### 3.4 批量 AI 工作流

- 未实现批处理任务队列、审核工作台、字段级采纳流水。

---

## 4. 当前已知限制

- PDF 元数据提取受文档版式影响，识别准确率并非 100%。
- `open-md` 依赖操作系统默认程序关联；不同机器环境行为可能存在差异。
- 命令行里出现中文乱码时，通常是终端编码显示问题，不影响数据库存储逻辑本身。

---

## 5. 快速验收清单（建议每次改动后复测）

1. 新建粗读记录并保存，确认弹窗 + 自动清空。
2. 上传 PDF，确认题目/作者/年份自动回填。
3. 在记录中心升级为精读，确认记录仍为同一条。
4. 修改枚举选项后，录入页下拉立即生效。
5. 导出所选与按筛选导出都可下载 `.xlsx`。
6. 新建或打开 MD 路径均可成功（含不存在文件自动创建场景）。
7. 切换深浅色并刷新页面后，主题保持。
8. 首页仪表盘展示最近阅读、精读候选、导出快捷入口与论文数量速度盘。
9. 首页速度盘指针在半圆范围内正常映射，不出现越界或倒插。
