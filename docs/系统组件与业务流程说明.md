# 系统组件（静态）与业务流程图（动态）说明

## 1. 系统组件（静态视图）

### 1.1 总体组件图（逻辑）

```mermaid
flowchart LR
  U[用户浏览器] --> FE[前端 UI
  frontend/index.html + src/main.js]
  FE --> API[Express API
  backend/src/server.js]
  API --> DB[(SQLite
  backend/data/litnotes.db)]
  API --> PDF[PDF解析器
  backend/src/pdf.js]
  API --> EXP[导出器
  backend/src/exporter.js]
  API --> FS[本地文件系统
  notes + exports]
```

### 1.2 前端组件职责

- 页面骨架：`frontend/index.html`
  - 侧边栏导航（首页/录入页/记录中心/设置）
  - 主内容容器（4 个 panel）
- 主交互：`frontend/src/main.js`
  - 状态管理（模板、枚举、记录、筛选、编辑状态）
  - 页面渲染与事件绑定
  - 首页仪表盘、录入、筛选、导出、模板管理、枚举管理
- API 客户端：`frontend/src/api/client.js`
  - 所有后端接口封装
- 主题：`frontend/src/theme/index.js`
  - 日夜模式切换与持久化
- 样式：`frontend/src/styles.css`
  - 侧边栏布局、卡片、表格、模态框、响应式规则

### 1.3 后端组件职责

- 应用入口/路由：`backend/src/server.js`
  - 静态资源托管
  - REST API
  - MD 创建/打开、导出下载、PDF 上传解析
- 数据层：`backend/src/db.js`
  - SQLite 初始化与迁移
  - 模板、枚举、设置、文献相关数据库操作
- 常量定义：`backend/src/constants.js`
  - 默认模板字段与默认枚举
- PDF 解析：`backend/src/pdf.js`
  - 标题/作者/年份/DOI 提取
  - 全文提取（AI full_pdf 场景）
- 导出器：`backend/src/exporter.js`
  - `.xlsx` 生成
- AI 调用与任务管理：`backend/src/server.js` + `backend/src/db.js`
  - 单篇分析、批量分析、任务轮询、会话重置

### 1.4 数据存储对象（核心表）

- `template_defs`：模板定义（粗读/精读、多套模板）
- `template_fields`：模板字段定义
- `enum_groups` + `enum_options`：下拉枚举
- `papers`：文献主数据（基础字段 + 阶段）
- `paper_field_values`：动态字段值（模板字段值）
- `paper_transitions`：粗读 -> 精读流转日志
- `exports`：导出记录
- `ai_jobs`：AI mock 任务记录
- `app_settings`：系统设置（如 notesDir）

## 2. 业务流程图（动态视图）

### 2.1 新建记录（含可选自动创建 MD）

```mermaid
flowchart TD
  A[录入页填写基础字段+模板字段] --> B{是否拖入PDF}
  B -- 是 --> C[调用 /api/pdf/parse]
  C --> D[回填题目/作者/年份等]
  B -- 否 --> E[继续填写]
  D --> E
  E --> F{autoCreateMd 且 mdPath为空?}
  F -- 是 --> G[后端创建MD文件]
  F -- 否 --> H[使用已有mdPath]
  G --> I[写入 papers + paper_field_values]
  H --> I
  I --> J[返回保存成功]
  J --> K[前端弹窗提示并清空表单]
```

### 2.2 首页仪表盘（最近阅读 / 精读候选 / 快捷导出）

```mermaid
flowchart TD
  A[用户进入首页 Home] --> B[前端读取 state.papers]
  B --> C[计算最近阅读 TopN]
  B --> D[计算精读候选 TopN]
  B --> E[计算总量/粗读/精读/候选率]
  E --> F[渲染速度盘可视化]
  C --> G{用户操作}
  D --> G
  G -- 编辑 --> H[跳转录入页并回填表单]
  G -- 升级精读 --> I[POST /api/papers/:id/upgrade]
  G -- 快捷导出 --> J[POST /api/export]
  I --> K[刷新 papers 并重绘首页]
  J --> L[返回 downloadUrl 并下载]
```

### 2.2A 录入页字段级 AI（含会话复用）

```mermaid
flowchart TD
  A[用户点击某字段 AI生成] --> B{是否已有同文献 aiSessionId}
  B -- 否 --> C[弹出全文发送确认]
  C --> D[POST /api/ai/analyze-and-fill 建立会话]
  B -- 是 --> E[复用 aiSessionId]
  E --> F[POST /api/ai/analyze-and-fill 按字段生成]
  D --> F
  F --> G[前端展示建议值]
  G --> H{用户是否采纳}
  H -- 是 --> I[写入表单并手动保存]
  H -- 否 --> J[跳过该字段]
```

### 2.3 记录中心筛选与导出

```mermaid
flowchart TD
  A[用户设置筛选条件] --> B[GET /api/papers]
  B --> C[渲染记录列表]
  C --> D{导出方式}
  D -- 导出所选 --> E[POST /api/export with selectedIds]
  D -- 导出筛选结果 --> F[POST /api/export with filters]
  E --> G[生成xlsx到 backend/data/exports]
  F --> G
  G --> H[返回 downloadUrl]
  H --> I[前端打开下载链接]
```

### 2.3A 记录中心批量 AI 分析

```mermaid
flowchart TD
  A[用户在记录中心勾选多条记录] --> B[点击 AI分析所选]
  B --> C[确认全文发送]
  C --> D[POST /api/ai/batch/analyze]
  D --> E[返回 batchJobId]
  E --> F[前端轮询 GET /api/ai/jobs/:id]
  F --> G{任务结束?}
  G -- 否 --> F
  G -- 是 --> H[展示每条成功/失败]
  H --> I{逐条应用建议?}
  I -- 是 --> J[PUT /api/papers/:id source=ai_applied]
  I -- 否 --> K[跳过]
```

### 2.4 粗读升级精读

```mermaid
flowchart TD
  A[记录中心点击 升级精读] --> B[POST /api/papers/:id/upgrade]
  B --> C{当前是否已是deep}
  C -- 是 --> D[直接返回当前记录]
  C -- 否 --> E[更新 papers.stage=deep]
  E --> F[写入 paper_transitions]
  F --> G[返回更新后的记录]
  G --> H[前端刷新列表]
```

### 2.5 打开 MD（含缺失文件自动创建）

```mermaid
flowchart TD
  A[记录中心点击 打开MD] --> B[POST /api/papers/open-md]
  B --> C{mdPath 合法且扩展名为 md/markdown?}
  C -- 否 --> D[返回 400 错误]
  C -- 是 --> E{文件存在?}
  E -- 否 --> F[创建目录并生成md模板文件]
  E -- 是 --> G[直接打开]
  F --> G
  G --> H{open库调用成功?}
  H -- 是 --> I[返回 ok]
  H -- 否且Windows --> J[cmd /c start 回退打开]
  J --> I
  H -- 否且非Windows --> K[返回 500 错误]
```

## 3. 关键时序（前后端交互）

### 3.1 页面初始化时序

```mermaid
sequenceDiagram
  participant U as 用户
  participant FE as 前端 main.js
  participant API as 后端 server.js
  participant DB as SQLite

  U->>FE: 打开页面
  FE->>API: GET /api/templates
  API->>DB: 查询模板与字段
  DB-->>API: 返回

  FE->>API: GET /api/enums
  API->>DB: 查询枚举
  DB-->>API: 返回
  API-->>FE: 返回枚举

  FE->>API: GET /api/settings
  FE->>API: GET /api/papers
  API->>DB: 查询设置/记录
  DB-->>API: 返回
  API-->>FE: 返回数据
  FE-->>U: 渲染录入页与列表
```

## 4. 非功能边界说明

- 运行形态：本地单机（`127.0.0.1:3000`）。
- PDF 处理：仅本地内存处理，不上传云端。
- MD 打开：依赖操作系统默认应用关联。
- 首页速度盘为前端计算型可视化，当前不引入额外统计接口。
- 数据规模：当前按轻量本地使用设计，未做分布式与多租户扩展。
